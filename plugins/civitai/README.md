# Civitai Plugin

Downloads models, images and previews from [Civitai](https://civitai.com).

## Domains

- `civitai.com`

## Usage

1. Install from **Plugins → Repositorio** in Ørion Gallery Downloader
2. Add a supported Civitai URL to the queue
3. The plugin engine will handle probing and downloads

## Supported URLs

- `https://civitai.com/models/*`
- `https://civitai.com/user/*/images`

## Notes

- Model pages download version files plus up to five preview images per version.
- User image pages paginate through the public Civitai image API.
